# Surya Flowers — static website

This branch contains a simple static website for Surya Flowers (Eluru). Files were added to the `website` branch.

How to publish (GitHub Pages):
1. Go to Settings → Pages in your repository.
2. Select the `website` branch and `/ (root)` folder, then Save.
3. Your site will be available at https://<your-github-username>.github.io/<repo-name>/ after a few minutes.

Notes:
- The site uses placeholder SVG images in the `assets/` folder. Replace them with real photos by uploading images to `assets/` and adjusting file names in `index.html`.
- WhatsApp order button sends a message to +91 93815 26635.
